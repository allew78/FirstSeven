Jordbruket och EU

Sverige är medlem i Europeiska unionen, EU. Inom EU finns många gemensamma regler för hur jordbruket ska bedrivas. Medlemsländerna betalar avgifter till EU, och under årens lopp har mycket av de pengarna gått till stöd åt jordbrukarna i de olika länderna för att de skulle fortsätta att odla som de brukat göra. Men numera handlar det mera om att ge stöd så att landsbygden kan utvecklas. Bönderna ska kunna tjäna pengar genom att bli effektivare, och jordbruket måste moderniseras i alla delar av EU.

Ungdomar kan söka bidrag för att starta ett eget jordbruk. Det är ju dyrt med alla maskiner som behövs redan från början. Bönder som vill kombinera sitt jordbruk med någon verksamhet som lockar turister kan också söka bidrag. Man kan också söka stöd för att utbilda sig för att odla ekologiskt. Det behövs ny kunskap för att driva ett ekologiskt jordbruk.
