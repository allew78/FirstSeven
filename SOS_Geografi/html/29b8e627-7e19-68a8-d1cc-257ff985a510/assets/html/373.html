Transporter

Troligen kommer mer tranporter att gå med järnväg i framtiden. Även eldrivna fordon kommer att bli vanligare, precis som hybrider som kan gå på både el och andra bränslen. Men då måste elnäten byggas ut och produktionen av el öka kraftigt. Det måste också byggas upp ett system med laddningsstationer där elfordonen kan ”tanka”.
