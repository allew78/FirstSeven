Naturkatastrofer

En stor del av jordens befolkning bor i områden som mer eller mindre ofta drabbas av naturkatastrofer som jordbävningar, vulkanutbrott, torka eller översvämningar.

Torra områden

I utkanterna av jordens ökenområden är livet helt beroende av det lilla regn som ändå brukar falla. De flesta år kommer det tillräckligt för att man ska kunna odla sina grödor, men ibland inträffar perioder då det regnar mindre än normalt, och ibland kan det gå väldigt lång tid då det inte regnar alls. Människorna som bor där är förstås väldigt sårbara.

Genom att bygga terrasser i sluttningarna kan bönderna förhindra att jorden spolas bort när det regnar. Vattnet kan istället samlas upp i kanaler och användas i odlingarna.

Men en hel del kan göras för att förbättra situationen. Brunnar kan borras, bevattningskanaler kan byggas och jordbruket kan anpassas bättre till det torra klimatet. Och med internationell hjälp kan lager med livsmedel byggas upp för att användas när torkan slår till.
