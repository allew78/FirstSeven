Människorna påverkar vattnets kretslopp

I landskapet på bilden finns inga människor. Men nu tänker vi oss att det också finns samhällen där med människor som lever och arbetar på åkrar, i fabriker och på kontor, i skolor, affärer och på sjukhus.

De använder vatten från floderna och sjön och de borrar nog också brunnar för att hämta upp grundvatten. Där floderna forsar ner för sluttningarna bygger de vattenkraftverk för att få elektricitet.

Med hjälp av starka maskiner kan de göra stora ingrepp som förändrar landskapet när de vill komma åt mer vatten. Avlopp från deras bostäder, fabriker och jordbruk smutsar ner vattnet om det inte renas noggrant. Ibland börjar olika grupper att bråka om vattnet. Längre fram i kapitlet kan du läsa mer om sådana problem.
